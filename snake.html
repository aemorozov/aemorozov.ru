Snake_1.5

This is my first real game

The game itself was written based on a square field drawn in one of the lessons on JS. After the lesson, it seemed to me that it would be great to implement a simple game on such a field, which I did.

Control of the game is implemented both from a laptop and from a touchscreen. In this version of the game, the snake accelerates with each caught target.
